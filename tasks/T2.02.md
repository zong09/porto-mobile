## Facts you need (dependency signatures — do NOT open their files)
- `portfolioRepoProvider` → `PortfolioRepo` with: `create({required String name, required int color})→Future<Portfolio>`,
  `save(Portfolio)→Future<void>`, `remove(String id)→Future<void>`, `reorder(List<String>)→Future<void>`,
  `all()→Future<List<Portfolio>>`.
- `assetRepoProvider` → `AssetRepo` with: `create({required String portfolioId, required String type,
  required String symbol, required String name, required String currency, String? cgId,
  String? yahooSymbol, double? manualPrice, String direction='long'})→Future<Asset>`,
  `save(Asset)→Future<void>`, `remove(String id)→Future<void>`, `allFor(String portfolioId)→Future<List<Asset>>`.
- `transactionRepoProvider` → `TransactionRepo` with `byAsset(String assetId)→Future<List<Transaction>>`.
- `PositionCalculator.calculate(List<TxInput> txs, {String direction='long'})→PositionSummary`
  (from `../domain/position_calculator.dart`). `TxInput({required double quantity, required double price,
  double fee=0, required String side, String? date})`. `PositionSummary` has fields
  `quantity, avgCost, totalCost, realizedPnl` (all double) + `direction` (String).
- Drift rows `Portfolio`(id,name,color,sortOrder) and `Asset`(id,portfolioId,type,symbol,name,currency,
  cgId?,yahooSymbol?,manualPrice?,direction,sortOrder) and `Transaction`(id,assetId,side,quantity,price,
  fee,date,createdAt) — all from `../db/database.dart`. Enum columns are plain `String`.
- Providers `portfolioRepoProvider`/`assetRepoProvider`/`transactionRepoProvider` are in the repo files
  (`../repos/portfolio_repo.dart`, `../repos/asset_repo.dart`, `../repos/transaction_repo.dart`) — import those.
- The generated notifier provider will be **`portfoliosProvider`** (generator strips `Notifier`).

## Implementation (`state/portfolios_notifier.dart`)
```dart
import 'package:freezed_annotation/freezed_annotation.dart';
import 'package:riverpod_annotation/riverpod_annotation.dart';
import '../db/database.dart';
import '../domain/position_calculator.dart';
import '../repos/portfolio_repo.dart';
import '../repos/asset_repo.dart';
import '../repos/transaction_repo.dart';
part 'portfolios_notifier.freezed.dart';
part 'portfolios_notifier.g.dart';

@freezed
abstract class AssetNode with _$AssetNode {         // MUST be abstract
  const factory AssetNode({required Asset asset, required PositionSummary position}) = _AssetNode;
}
@freezed
abstract class PortfolioNode with _$PortfolioNode {
  const factory PortfolioNode({required Portfolio portfolio, required List<AssetNode> assets}) = _PortfolioNode;
}
@freezed
abstract class PortfoliosState with _$PortfoliosState {
  const factory PortfoliosState({required List<PortfolioNode> nodes}) = _PortfoliosState;
}

@riverpod
class PortfoliosNotifier extends _$PortfoliosNotifier {
  PortfolioRepo get _pRepo => ref.read(portfolioRepoProvider);
  AssetRepo get _aRepo => ref.read(assetRepoProvider);
  TransactionRepo get _tRepo => ref.read(transactionRepoProvider);

  @override
  Future<PortfoliosState> build() async {
    final ports = await _pRepo.all();
    final nodes = <PortfolioNode>[];
    for (final p in ports) {
      final assets = await _aRepo.allFor(p.id);
      final assetNodes = <AssetNode>[];
      for (final a in assets) {
        final txs = await _tRepo.byAsset(a.id);
        final inputs = txs.map((t) => TxInput(
          quantity: t.quantity, price: t.price, fee: t.fee, side: t.side, date: t.date)).toList();
        final pos = PositionCalculator.calculate(inputs, direction: a.direction);
        assetNodes.add(AssetNode(asset: a, position: pos));
      }
      nodes.add(PortfolioNode(portfolio: p, assets: assetNodes));
    }
    return PortfoliosState(nodes: nodes);
  }

  Future<void> _reload() async { ref.invalidateSelf(); await future; }

  Future<void> addPortfolio({required String name, required int color}) async {
    await _pRepo.create(name: name, color: color); await _reload();
  }
  Future<void> renamePortfolio(String id, String name) async {
    final p = (await _pRepo.all()).firstWhere((x) => x.id == id);
    await _pRepo.save(p.copyWith(name: name)); await _reload();
  }
  Future<void> recolorPortfolio(String id, int color) async {
    final p = (await _pRepo.all()).firstWhere((x) => x.id == id);
    await _pRepo.save(p.copyWith(color: color)); await _reload();
  }
  Future<void> deletePortfolio(String id) async { await _pRepo.remove(id); await _reload(); }
  Future<void> reorderPortfolios(List<String> ids) async { await _pRepo.reorder(ids); await _reload(); }

  Future<void> addAsset({required String portfolioId, required String type, required String symbol,
      required String name, required String currency, String? cgId, String? yahooSymbol,
      double? manualPrice, String direction = 'long'}) async {
    await _aRepo.create(portfolioId: portfolioId, type: type, symbol: symbol, name: name,
      currency: currency, cgId: cgId, yahooSymbol: yahooSymbol, manualPrice: manualPrice,
      direction: direction);
    await _reload();
  }
  Future<void> saveAsset(Asset asset) async { await _aRepo.save(asset); await _reload(); }
  Future<void> deleteAsset(String id) async { await _aRepo.remove(id); await _reload(); }
}
```

## Test (`test/portfolios_notifier_test.dart`) — §10.4 harness, FRESH db+container per test (`setUp`/`tearDown`)
Imports: `package:drift/native.dart`, `package:flutter_riverpod/flutter_riverpod.dart`,
`package:flutter_test/flutter_test.dart`, `package:porto_mobile/src/db/database.dart`,
`package:porto_mobile/src/state/providers.dart`, `package:porto_mobile/src/state/portfolios_notifier.dart`.
Helpers: `n() => container.read(portfoliosProvider.notifier);`
`state() async => await container.read(portfoliosProvider.future);`
1. empty: `state().nodes` empty.
2. `n().addPortfolio(name:'Main', color:2)` → nodes length 1, first.portfolio.name=='Main', assets empty.
3. add two portfolios; `reorderPortfolios([id2,id1])` → nodes[0].portfolio.id==id2.
4. `renamePortfolio(id,'X')` → nodes[0].portfolio.name=='X'; `recolorPortfolio(id,4)` → color==4.
5. position: add portfolio; `addAsset(portfolioId:pid, type:'crypto', symbol:'BTC', name:'Bitcoin', currency:'USD')`;
   then add txs directly on the asset via the notifier is not possible (no tx method here) — instead insert
   two buy txs through the DB: get assetId from `state().nodes[0].assets[0].asset.id`, then
   `await container.read(transactionRepoProvider).add(assetId: aid, side:'buy', quantity:2, price:100, date:'2026-01-01');`
   and another `quantity:1, price:200`. Re-read state → `nodes[0].assets[0].position.quantity` closeTo 3,
   `.totalCost` closeTo 400.
6. `deletePortfolio(id)` → nodes empty (cascade removes assets/txs too).
7. currency lock via notifier: after case-5 asset exists, `saveAsset(asset.copyWith(currency:'THB'))`
   throws (expect `throwsA(isA<ArgumentError>())`).
