## What to build
### `LiabilitiesScreen` — `ConsumerWidget` watching `liabilitiesProvider`
Per design-liabilities "Liabilities (list)": back button (pops to Overview); title "หนี้สิน";
"＋ เพิ่มหนี้สิน" (opens create sheet); total "หนี้สินรวม" = Σ `liability.amount` (note: mixed
currencies are summed raw here — no FX in this state; acceptable for the screen). Each liability =
`PlainCard`/`ListRowTile`: leading 42×42 `#F6E4EA` square with `−` `#A84E71`; title `name`; subtitle
currency; trailing `Formatters.money(amount, currency: liability.currency)` color `#A8341C`;
`onTap` → open Adjust sheet. Dashed "＋ เพิ่มหนี้สินใหม่" card. Empty state: centered muted note.

### `LiabilitySheet` (`liability_sheet.dart`) — both adjust and create

- **Adjust** (design-liabilities "Adjust sheet"): current-balance banner (`liability.amount`);
  ประเภท segmented toggle **จ่าย (pay)** vs **เพิ่ม (add)** (local state); จำนวนเงิน amount input
  (`> 0`); วันที่ date field (default today = `DateTime.now().toIso8601String().substring(0,10)`).
  Save → `ref.read(liabilitiesProvider.notifier).adjust(liabilityId: liability.id, type: <pay|add>,
  amount: amt, date: date)`.
- **Create**: ชื่อ (non-empty) + จำนวนเงิน (>0) + สกุลเงิน segmented (THB/USD). Save →
  `addLiability(name:…, amount:…, currency:…)`.

## Fake-state test pattern (`test/liabilities_screen_test.dart`)
```dart
class _FakeLiab extends LiabilitiesNotifier {
  final LiabilitiesState _s; final List<String> calls = [];
  _FakeLiab(this._s);
  @override FutureOr<LiabilitiesState> build() => _s;
  @override Future<void> adjust({required String liabilityId, required String type,
      required double amount, required String date}) async { calls.add('$type:$amount'); }
}
```
(Override `adjust` in the fake to RECORD instead of hitting a DB, so the flow test asserts the call.)
Tests:
1. `testWidgets('renders one card per liability + total')`: 2 liabilities → both names found, total
   shown; no exception.
2. `testWidgets('empty state note when no liabilities')`: empty list → the empty note is found.
3. `testWidgets('adjust pay records adjust(pay)')`: pump `LiabilityAdjustSheet(liability: L)` with the
   recording fake; choose **จ่าย**, enter amount, tap save; assert `fake.calls` contains a `pay:` entry.
4. `testWidgets('adjust add records adjust(add)')`: same but **เพิ่ม** → `add:` entry recorded.

> To inject the recording fake, override `liabilitiesProvider.overrideWith(() => fake)` and read
> `container.read(liabilitiesProvider.notifier)` — but for a **widget** test just build the sheet
> inside a `ProviderScope(overrides:[liabilitiesProvider.overrideWith(() => fake)])`.
