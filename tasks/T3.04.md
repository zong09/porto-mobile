# T3.04 — Shared widgets: cards + sheet shell   (Owner: Qwen)

## Goal
The reusable presentational widgets every Phase-3 screen composes: card containers, the Overview
stat card, a generic list row, and the bottom-sheet shell (grab handle + title + close). Pure
presentation — no state, no DB, no network.

- `lib/src/ui/widgets/cards.dart`

## cards.dart

```dart
/// White rounded container. radius 18, padding 15×16 by default.
class PlainCard extends StatelessWidget {
  final Widget child;
  final EdgeInsetsGeometry padding;
  const PlainCard({super.key, required this.child,
      this.padding = const EdgeInsets.symmetric(horizontal: 16, vertical: 15)});
  // Container: color AppColors.surface, borderRadius 18, padding, child.
}

/// A card that lays out rows with a 1px #F5EDDE divider between each (not after the last).
/// Card padding is 4×15 (rows provide their own vertical padding of 12).
class DividedCard extends StatelessWidget {
  final List<Widget> rows;
  const DividedCard({super.key, required this.rows});
  // Column of rows interleaved with Container(height:1, color Color(0xFFF5EDDE)).
}

/// Overview stat tile. Expands to fill (wrap in Expanded/Row by caller).
class StatCard extends StatelessWidget {
  final String label;
  final String value;
  final Color? valueColor; // default AppColors.text
  const StatCard({super.key, required this.label, required this.value, this.valueColor});
  // Container: surface, radius 16, padding h13 v11; Column(cross start):
  //   Text(label, size 10.5, color muted); Text(value, size 15, w700, tabular, color valueColor??text).
}

/// Generic list row: leading avatar/icon, title+subtitle, right-aligned trailing.
class ListRowTile extends StatelessWidget {
  final Widget leading;            // e.g. a 40×40 tinted square (caller builds)
  final String title;
  final String? subtitle;
  final Widget trailing;           // caller builds the value/sub column or chip
  final VoidCallback? onTap;
  const ListRowTile({super.key, required this.leading, required this.title,
      this.subtitle, required this.trailing, this.onTap});
  // Row(align center, gap 12 via SizedBox/Padding): leading, Expanded(Column: title 14/w700,
  //   subtitle 11 color muted2), trailing. Vertical padding 12. Wrap in InkWell if onTap != null.
}
```
Use tabular figures for numeric values: `TextStyle(fontFeatures: [FontFeature.tabularFigures()])`
(or reuse `AppType.numTabular`). Muted2 = `AppColors.muted2` (#A89A86).

## sheet_shell.dart

```dart
/// The chrome of a Porto bottom sheet: grab handle, title row with a close button, then child.
class SheetShell extends StatelessWidget {
  final String title;
  final Widget child;
  final VoidCallback? onClose;
  const SheetShell({super.key, required this.title, required this.child, this.onClose});
  // Container: color AppColors.bg (#FAF5EC), borderRadius vertical top 30, padding (14,14,20 h; 46 bottom).
  // Column(min): grab handle (40×5, radius 999, color #E2D6C2, centered),
  //   SizedBox(14), Row: Text(title, size 17, w700) + Spacer + close circle
  //     (28×28, color #F0E7D6, child Text('✕', size 13, color muted), InkWell→onClose),
  //   SizedBox(14), child.
}

/// Presents [builder] as a modal bottom sheet with the Porto scrim + rounded top.
Future<T?> showPortoSheet<T>(BuildContext context, {required String title,
    required WidgetBuilder builder}) {
  return showModalBottomSheet<T>(
    context: context,
    isScrollControlled: true,
    backgroundColor: Colors.transparent,
    barrierColor: const Color(0x73291C12), // rgba(41,28,18,0.45)
    builder: (ctx) => SheetShell(title: title, onClose: () => Navigator.of(ctx).pop(),
        child: builder(ctx)),
  );
}
```

## Test (`test/shared_widgets_test.dart`)
Wrap each pump in `MaterialApp(home: Scaffold(body: ...))`.
1. `testWidgets('PlainCard renders child')`: pump `PlainCard(child: Text('hi'))`; `find.text('hi')` one.
2. `testWidgets('DividedCard inserts dividers')`: pump `DividedCard(rows: [Text('a'), Text('b'), Text('c')])`;
   expect all three texts found and `expect(tester.takeException(), isNull)`.
3. `testWidgets('StatCard shows label + value')`: pump
   `Row(children: [Expanded(child: StatCard(label: 'Assets', value: '฿4.67M'))])`;
   `find.text('Assets')` and `find.text('฿4.67M')` each one.
4. `testWidgets('ListRowTile shows title/subtitle and fires onTap')`: pump a `ListRowTile` with
   `leading: SizedBox(width: 40, height: 40)`, `title: 'Crypto'`, `subtitle: 'BTC·ETH'`,
   `trailing: Text('฿1,842,300')`, `onTap` toggling a bool; tap it, assert bool flipped and texts found.
5. `testWidgets('SheetShell shows title, close fires onClose')`: pump
   `SheetShell(title: 'เพิ่มรายการ', onClose: ..., child: Text('body'))`; find title + 'body';
   tap the '✕' and assert onClose fired.
