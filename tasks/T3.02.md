## API to build
```dart
class DonutSlice {
  final double value;
  final Color color;
  const DonutSlice(this.value, this.color);
}

/// A ring donut. Slices are drawn clockwise starting at 12 o'clock.
class DonutChart extends StatelessWidget {
  final List<DonutSlice> slices;
  final double strokeFraction; // ring thickness / radius, default 0.34
  final double gapDegrees;     // gap between slices, default 0
  const DonutChart(this.slices, {super.key, this.strokeFraction = 0.34, this.gapDegrees = 0});
  @override
  Widget build(BuildContext context) => CustomPaint(
      painter: DonutChartPainter(slices, strokeFraction: strokeFraction, gapDegrees: gapDegrees),
      child: const SizedBox.expand());
}

class DonutChartPainter extends CustomPainter { ... }
```

## Algorithm
Let `w = size.width`, `h = size.height`, `radius = min(w,h)/2`,
`center = Offset(w/2, h/2)`, `stroke = radius * strokeFraction`.
1. If slices empty or `total <= 0` → paint nothing, return.
2. `total = sum(values)`; `gap = gapDegrees * π/180` (radians).
3. `start = -π/2` (12 o'clock). For each slice:
   `sweep = (value/total) * 2π`; draw an arc on `Rect.fromCircle(center, radius - stroke/2)`
   with `Paint()..style = stroke..strokeWidth = stroke..color = slice.color`
   (`strokeCap = gap>0 ? round : butt`); if `gap>0`, inset the arc by `gap/2` on each side
   (`startAngle = start + gap/2`, `sweepAngle = sweep - gap`). Advance `start += sweep`.
4. `canvas.drawArc(rect, startAngle, sweepAngle, false, paint)` (useCenter = false → ring).
5. `shouldRepaint`: true if slices (value+color), strokeFraction, or gapDegrees differ.

1. `testWidgets('three-slice donut paints')`: pump
   `SizedBox(width: 64, height: 64, child: DonutChart(const [DonutSlice(50, Color(0xFFEC6530)), DonutSlice(26, Color(0xFFFFC79A)), DonutSlice(24, Color(0xFFFFE3CF))]))`

   `DonutChart(const [DonutSlice(0, Color(0xFFEC6530))])`; no exception.

   `expect(DonutChartPainter(const [DonutSlice(1, Color(0xFFEC6530))]).shouldRepaint(DonutChartPainter(const [DonutSlice(2, Color(0xFFEC6530))])), isTrue);`
