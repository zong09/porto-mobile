## Algorithm (port EXACTLY)
```
summary({assets, liabilities, fx}):
  totalLiabilitiesThb = sum over liabilities of amount * (currency=='USD' ? fx : 1)
  totalAssetsThb=0; totalCostThb=0; todayPlThb=0
  for a in assets:
    pos = PositionCalculator.calculate(a.txs, direction: a.direction)
    if pos.quantity <= 0: continue
    // effective price
    if a.type=='deposit': price=1.0
    else if a.type=='fund': price = a.manualPrice ?? 0.0
    else: price = a.price
    multiplier = a.currency=='USD' ? fx : 1
    assetValThb = pos.quantity * price * multiplier
    isShort = a.direction=='short'
    totalAssetsThb += isShort ? -assetValThb : assetValThb
    totalCostThb += pos.quantity * pos.avgCost * multiplier
    if a.chg24h != 0:
      rawPl = assetValThb - assetValThb/(1 + a.chg24h/100)
      todayPlThb += isShort ? -rawPl : rawPl
  netWorthThb = totalAssetsThb - totalLiabilitiesThb
  return NetWorthSummary(totalAssetsThb, totalLiabilitiesThb, netWorthThb, todayPlThb, totalCostThb, fx)
```
