## Dio usage rules
- `doYahooRequest` sets header `User-Agent` (browser UA string below) and, when provided, `Cookie`.
- Use default validateStatus (throws on non-2xx) so a 401/403 direct call is caught and triggers crumb retry.
- Chart JSON: `resp.data` is a decoded `Map`. The crumb endpoint returns **plain text** — request it with `Options(responseType: ResponseType.plain)` and read `resp.data as String`.

## Constants
- `UA = 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36'`
- `SYMBOL_REGEX = RegExp(r'^[A-Za-z0-9.\-^=]{1,15}$')`

## Algorithm (port EXACTLY)
```
getStockPrice(symbol):
  validate symbol against SYMBOL_REGEX; invalid → throw ApiError('invalid symbol: $symbol')
  data = await fetchYahooChart(symbol, '1d', '1d')
  if data != null:
    meta = data['chart']?['result']?[0]?['meta']
    if meta != null and meta['regularMarketPrice'] != null:
      price = (meta['regularMarketPrice'] as num).toDouble()
      prev  = (meta['chartPreviousClose'] ?? meta['previousClose'] ?? meta['regularMarketPrice']) as num
      chg   = prev != 0 ? (price / prev - 1) * 100 : 0.0
      return StockQuote(price, chg)
  throw ApiError('Failed to fetch stock price for $symbol')

getFxRate():
  try: return (await getStockPrice('THB=X')).price
  catch: return 35.84

fetchYahooChart(symbol, range, interval):        # returns Map or null
  try:
    r = await doYahooRequest(symbol, range, interval)
    if r != null: return r
  catch: (log/ignore)
  try:
    if _crumb == null: await refreshYahooCredentials()
    if _crumb != null:
      r = await doYahooRequest(symbol, range, interval, crumb: _crumb, cookie: _cookie)
      if r != null: return r
  catch: (ignore)
  return null

doYahooRequest(symbol, range, interval, {crumb, cookie}):
  url = 'https://query1.finance.yahoo.com/v8/finance/chart/' + Uri.encodeComponent(symbol)
        + '?range=$range&interval=$interval'
  if crumb != null: url += '&crumb=' + Uri.encodeComponent(crumb)
  headers = {'User-Agent': UA}; if cookie != null: headers['Cookie'] = cookie
  resp = dio.get(url, options: Options(headers: headers))   # throws on non-2xx
  return resp.data                                          # decoded Map

refreshYahooCredentials():                        # sets _cookie, _crumb (nullable fields)
  try:
    fc = dio.get('https://fc.yahoo.com', options: Options(headers:{'User-Agent':UA}, validateStatus:(_)=>true))
    setCookie = fc.headers.map['set-cookie']         # List<String>? ; take first, keep part before ';'
    if setCookie not empty: _cookie = setCookie[0].split(';')[0]
    ch = dio.get('https://query2.finance.yahoo.com/v1/test/getcrumb',
                 options: Options(headers:{'User-Agent':UA, if _cookie!=null 'Cookie':_cookie},
                                  responseType: ResponseType.plain, validateStatus:(_)=>true))
    if ch.statusCode == 200: _crumb = ch.data as String
  catch: (ignore)
```

## Test vectors (`test/yahoo_client_test.dart`) — `class MockDio extends Mock implements Dio {}`
`setUpAll(() => registerFallbackValue(Options()));`
Match endpoints by URL substring: chart = `contains('/v8/finance/chart/')`, crumb =
`contains('getcrumb')`, cookie = `contains('fc.yahoo.com')`.

1. **Direct stock success**: chart call → 200, data =
   `{'chart':{'result':[{'meta':{'regularMarketPrice':100.0,'chartPreviousClose':90.0}}]}}`.
   `getStockPrice('AAPL')` → `price==100.0`, `chg` closeTo((100/90-1)*100, 0.01).
2. **401 → crumb → retry**: first chart call throws `DioException` (statusCode 401);
   `fc.yahoo.com` → Response with `headers` `{'set-cookie':['A=B; path=/']}`; `getcrumb` → 200 plain `'CR'`;
   second chart call (`options` containing crumb) → 200 with the meta from case 1.
   `getStockPrice('AAPL')` → `price==100.0`. (Stub the chart via a call counter: 1st throws, 2nd returns.)
3. **FX from THB=X**: chart for `THB=X` → 200 `{'chart':{'result':[{'meta':{'regularMarketPrice':35.5,'chartPreviousClose':35.5}}]}}`.
   `getFxRate()` → `35.5`.
4. **FX fallback**: chart call always throws → `getFxRate()` returns `35.84`.
5. **.BK passthrough**: `getStockPrice('PTT.BK')` with a `captureAny()` on the URL asserts the captured URL `contains('PTT.BK')`.
6. **Invalid symbol**: `getStockPrice('AA!!')` throws `ApiError`.
