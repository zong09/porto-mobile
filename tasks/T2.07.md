## Implementation (`prices/price_repository.dart`)
```dart
import 'package:dio/dio.dart';
import 'package:drift/drift.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import '../db/database.dart';
import '../db/misc_dao.dart';
import 'binance_client.dart';
import 'yahoo_client.dart';
import '../state/providers.dart';

class ResolvedPrice {
  final double price; final double chg24h; final String source;
  const ResolvedPrice(this.price, this.chg24h, this.source);
}

class PriceRepository {
  final MiscDao dao; final BinanceClient binance; final YahooClient yahoo;
  final DateTime Function() now;
  PriceRepository(this.dao, this.binance, this.yahoo, {DateTime Function()? now})
      : now = now ?? DateTime.now;

  Future<ResolvedPrice> resolve(Asset asset) async {
    if (asset.type == 'deposit') return const ResolvedPrice(1, 0, 'fixed');
    if (asset.type == 'fund') return ResolvedPrice(asset.manualPrice ?? 0, 0, 'manual');

    final isCrypto = asset.type == 'crypto';
    final key = isCrypto ? 'crypto:${asset.symbol}' : 'stock:${asset.symbol}';
    final ttl = isCrypto ? const Duration(seconds: 60) : const Duration(seconds: 90);

    final cached = await dao.getPrice(key);
    if (cached != null &&
        now().difference(DateTime.parse(cached.fetchedAt)) < ttl) {
      return ResolvedPrice(cached.price, cached.chg24h, 'cache');
    }

    try {
      double native; double chg;
      if (isCrypto) {
        final cp = (await binance.getPrices([asset.symbol]))[asset.symbol];
        if (cp == null) throw StateError('no price');
        native = asset.currency == 'USD' ? cp.usd : cp.thb;
        chg = asset.currency == 'USD' ? cp.usdChg : cp.thbChg;
      } else {
        final ysym = asset.yahooSymbol ??
            (asset.type == 'th' ? '${asset.symbol}.BK' : asset.symbol);
        final q = await yahoo.getStockPrice(ysym);
        native = q.price; chg = q.chg;
      }
      await dao.putPrice(PriceCacheCompanion.insert(
        key: key, price: native, chg24h: Value(chg),
        currency: asset.currency, fetchedAt: now().toUtc().toIso8601String()));
      return ResolvedPrice(native, chg, 'live');
    } catch (_) {
      if (cached != null) return ResolvedPrice(cached.price, cached.chg24h, 'cache');
      if (asset.manualPrice != null) return ResolvedPrice(asset.manualPrice!, 0, 'manual');
      return const ResolvedPrice(0, 0, 'none');
    }
  }
}

final priceRepositoryProvider = Provider((ref) => PriceRepository(
    ref.watch(miscDaoProvider),
    BinanceClient(Dio(), getFx: () async => YahooClient(Dio()).getFxRate()),
    YahooClient(Dio())));
```

  String? yahooSymbol}) => Asset(id:'x', portfolioId:'p', type:type, symbol:symbol, name:'n',
  currency:currency, cgId:null, yahooSymbol:yahooSymbol, manualPrice:manualPrice, direction:'long', sortOrder:0);`
Per test: `db = AppDatabase(NativeDatabase.memory()); dao = MiscDao(db);` fresh; `tearDown(db.close)`.
Fixed clock: `final t0 = DateTime.utc(2026, 1, 1, 12, 0, 0);`
1. deposit → `resolve(asset(type:'deposit'))` returns price 1, source 'fixed' (no client calls).
2. fund → `resolve(asset(type:'fund', manualPrice:12.5))` price 12.5, source 'manual'.
3. crypto live: mockBinance `getPrices(['BTC'])` → `{'BTC': CryptoPrice(usd:100, usdChg:5, thb:3600, thbChg:5)}`.
   repo = `PriceRepository(dao, mockBinance, mockYahoo, now: () => t0)`. `resolve(asset(type:'crypto', currency:'USD'))`
   → price 100, chg 5, source 'live'. AND `dao.getPrice('crypto:BTC')` is now non-null (persisted).
4. crypto fresh cache: pre-put cache `crypto:BTC` price 999 chg 1 fetchedAt `t0.toIso8601String()`,
   clock `() => t0.add(Duration(seconds: 30))` (< 60s) → resolve returns price 999, source 'cache';
   `verifyNever(() => mockBinance.getPrices(any()))`.
5. stale → live: same cache but clock `t0.add(Duration(seconds: 120))` (>60s), mockBinance returns
   fresh {'BTC': CryptoPrice(usd:200,...)} → source 'live', price 200.
6. fallback to cache on error: pre-put cache price 50; clock stale; mockBinance `getPrices` throws
   `Exception('net')` → resolve returns price 50, source 'cache'.
7. stock native + .BK: `resolve(asset(type:'th', symbol:'PTT', currency:'THB'))`; expect
   `mockYahoo.getStockPrice('PTT.BK')` called (stub it → StockQuote(price:35, chg:-1)); price 35, source 'live'.
