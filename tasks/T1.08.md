## Algorithm (port EXACTLY ŌĆö this is the contract)
```
calculate(txs, {direction='long'}):
  sorted = copy of txs, stable-sorted so that when BOTH a.date and b.date are non-null,
           order by a.date.compareTo(b.date); otherwise keep original order.
  quantity=0.0; totalCost=0.0; realizedPnl=0.0
  for tx in sorted:
    q=tx.quantity; p=tx.price; f=tx.fee (0 if null)
    if direction=='short':
      if side=='sell':                      # sell opens short
        totalCost += q*p + f;  quantity += q
      else if side=='buy':                  # buy covers
        avgCost = quantity>0 ? totalCost/quantity : 0
        realizedPnl += q*(avgCost - p) - f
        totalCost -= avgCost*q;  quantity -= q
        if quantity < 1e-9: quantity=0; totalCost=0
    else:  # long
      if side=='buy' or side=='deposit':
        totalCost += q*p + f;  quantity += q
      else if side=='sell' or side=='withdraw':
        avgCost = quantity>0 ? totalCost/quantity : 0
        realizedPnl += q*(p - avgCost) - f
        totalCost -= avgCost*q;  quantity -= q
        if quantity < 1e-9: quantity=0; totalCost=0
  avgCost = quantity>0 ? totalCost/quantity : 0
  return PositionSummary(quantity, avgCost, totalCost, realizedPnl, direction)
```
Notes: use `double`. The `< 1e-9` reset is REQUIRED. `direction` string is echoed back in the result.
