## What to build
### `AppBottomNav` (`app_bottom_nav.dart`)
Floating capsule per design-components §1: 4 tab slots + center FAB.
```dart
class AppBottomNav extends StatelessWidget {
  final int currentIndex;                 // 0=Overview 1=Portfolios 2=Transactions 3=Settings
  final ValueChanged<int> onTap;          // tab taps (skips the FAB)

```
Icons are inline `CustomPaint`/`Icon` per design-components §1 (Overview circle, Portfolios 2×2,
Transactions 3-lines, Settings sliders). Active tab: `#FFAE6E` stroke + 4px dot; inactive
`rgba(250,245,236,0.55)`. FAB = 46×46 `#EC6530` circle with white `+`.

### `OverviewScreen` (`overview.dart`)
`ConsumerWidget` watching `overviewProvider`. Layout per design-overview §A/§B:
- `.when(loading: spinner, error: msg, data: (st) => ...)`.
- If `st.summary == null` → a simple empty/first-run placeholder (a centered "เริ่มใช้เลย" is fine).
- Hero (gradient container): "Net Worth" label; value = `Formatters.money(summary.netWorthThb,
  currency:'THB')` (import `formatters.dart`); gain pill from `summary.todayPlThb` / % ; **AreaChart**
  hero variant fed `st.history.map((h) => h.netWorthThb).toList()` (white line, 0x24FFFFFF fill,
  end dot); timeframe pills (static, `1D` active — no wiring needed).
- If `st.offline` → show the "as of" banner using `st.asOf` (design-overview §State mapping).
- Sheet: 3 `StatCard`s (Assets `compact(totalAssetsThb)`, Liabilities `compact(totalLiabilitiesThb)`
  color `#A8341C`, กำไรรวม `pct(...)` color `#177E81`); the **Liabilities** StatCard is wrapped so
  `onTap` calls a `VoidCallback? onOpenLiabilities` (default null → no-op) — real route in T4.1.
- Section header "พอร์ตของฉัน" + "ดูทั้งหมด"; a portfolio list is OPTIONAL here (Overview state has
  no per-portfolio breakdown) — render the stat/hero faithfully; a portfolio summary list can be a
  simple placeholder note. (The Portfolios tab, T3.06, owns the full list.)

## Fake-state test pattern (`test/overview_screen_test.dart`)
```dart
class _FakeOverview extends OverviewNotifier {
  final OverviewState _s; _FakeOverview(this._s);
  @override FutureOr<OverviewState> build() => _s;
}
Widget _app(OverviewState s) => ProviderScope(
  overrides: [overviewProvider.overrideWith(() => _FakeOverview(s))],
  child: const MaterialApp(home: OverviewScreen()));
```
Tests:
1. `testWidgets('renders net worth + area chart from fake state')`: build with a state whose
   `summary` has `netWorthThb: 4286450, todayPlThb: 18240, ...` and `history:[.. 8 points ..]`;
   pump; expect a formatted net-worth string is found and `find.byType(AreaChart)` findsOneWidget;
   `expect(tester.takeException(), isNull)`.
2. `testWidgets('offline shows as-of banner')`: state `offline: true, asOf: <iso>`; expect the
   banner text present.
3. `testWidgets('summary==null shows empty state')`: no exception; empty-state marker present.
4. `testWidgets('AppShell shows bottom nav with 4 tabs + FAB and switches tab')`: pump
   `ProviderScope(overrides:[...], child: MaterialApp(home: AppShell()))`; expect
   `find.byType(AppBottomNav)` one; tap tab index 1 and assert no exception (placeholder shows).
