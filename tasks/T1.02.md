This brief + `CONTRACTS.md` §2.

## Spec
Use Drift's Dart-table API. Money columns = `real()`. Text = `text()`. Int = `integer()`.
Nullable columns use `.nullable()`. Enums are stored as their wire string → plain `text()` column.
Defaults via `.withDefault(const Constant(...))`. Primary keys via `@override Set<Column> get primaryKey`.
Foreign keys via `.references(Target, #id, onDelete: KeyAction.cascade)`.

Table classes (names exact):
- `Portfolios` : text id [PK] · text name · integer color · integer sortOrder default 0
- `Assets` : text id [PK] · text portfolioId references Portfolios #id cascade · text type · text symbol ·
  text name · text currency default 'THB' · text cgId nullable · text yahooSymbol nullable ·
  real manualPrice nullable · text direction default 'long' · integer sortOrder default 0
- `Transactions` : text id [PK] · text assetId references Assets #id cascade · text side ·
  real quantity · real price · real fee default 0 · text date · text createdAt
- `Liabilities` : text id [PK] · text name · real amount · text currency default 'THB'
- `LiabilityTransactions` : text id [PK] · text liabilityId references Liabilities #id cascade ·
  text type · real amount · text date · text createdAt
- `NetWorthHistory` : text date [PK] · real totalAssetsThb · real totalLiabilitiesThb ·
  real netWorthThb · real fxRate nullable
- `PriceCache` : text key [PK] · real price · real chg24h default 0 · text currency · text fetchedAt
- `Settings` : text key [PK] · text value

Do NOT create the database class or DAOs — that is T1.03/T1.04+. This file only declares tables.
It will not generate code on its own (no `part`); it is consumed by `database.dart` in T1.03.
