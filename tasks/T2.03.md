## TransactionRepo (`repos/transaction_repo.dart`)
```
const _sides = {'buy','sell','deposit','withdraw'};
class TransactionRepo { final TransactionDao dao; TransactionRepo(this.dao);
  void _validate(side, quantity, price, fee, date):
    if !_sides.contains(side) -> ArgumentError('side');
    if quantity <= 0 -> ArgumentError('quantity');
    if price < 0 -> ArgumentError('price');
    if fee < 0 -> ArgumentError('fee');
    if date.trim().isEmpty -> ArgumentError('date');
  Future<Transaction> add({assetId, side, quantity, price, fee=0, date}):
    _validate(side,quantity,price,fee,date);
    id = const Uuid().v4(); createdAt = DateTime.now().toUtc().toIso8601String();
    await dao.upsert(TransactionsCompanion.insert(id:id, assetId:assetId, side:side,
      quantity:quantity, price:price, date:date, createdAt:createdAt, fee: Value(fee)));
    return Transaction(id:id, assetId:assetId, side:side, quantity:quantity, price:price,
      fee:fee, date:date, createdAt:createdAt);
  Future<void> save(Transaction t):
    _validate(t.side,t.quantity,t.price,t.fee,t.date); await dao.upsert(t.toCompanion(true));
  Future<void> remove(String id) => dao.delete(id);
  Future<List<Transaction>> byAsset(String assetId) => dao.byAsset(assetId);
  Future<List<Transaction>> all() => dao.all();
}
final transactionRepoProvider = Provider((ref) => TransactionRepo(ref.watch(transactionDaoProvider)));
```

## Test (`test/transaction_repo_test.dart`)
Fresh DB **per test** (`setUp`, not `setUpAll`): `db = AppDatabase(NativeDatabase.memory());
repo = TransactionRepo(TransactionDao(db));` and `tearDown(() => db.close());`.
**FK constraint**: `transactions.assetId` → `assets.id` with `foreign_keys` ON, so in `setUp` you MUST
seed a parent portfolio + assets `'a1'`/`'a2'` first, else inserts fail with SqliteException(787):
`await db.into(db.portfolios).insert(PortfoliosCompanion.insert(id:'p1', name:'P', color:0));`
`await db.into(db.assets).insert(AssetsCompanion.insert(id:'a1', portfolioId:'p1', type:'crypto', symbol:'BTC', name:'Bitcoin'));` (and `'a2'`).
1. `add(assetId:'a1', side:'buy', quantity:0.5, price:100, fee:1, date:'2026-01-01')` → returned tx
   has non-empty id + createdAt, quantity==0.5, fee==1. `byAsset('a1')` has 1 row.
2. `quantity:0` throws `ArgumentError`; `price:-1` throws; `side:'bogus'` throws; `date:''` throws.
3. `fee` defaults to 0 when omitted: `add(assetId:'a1', side:'sell', quantity:1, price:2, date:'2026-01-02')`
   → its `fee == 0`.
4. `save(tx.copyWith(quantity: 2))` → `byAsset` shows quantity 2 (one row, not two).
5. `remove(id)` → `byAsset('a1')` no longer contains it.
6. `all()` returns rows across different assetIds.
