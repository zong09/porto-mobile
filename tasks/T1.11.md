class CryptoPrice {

## Dio usage rules (so non-2xx does NOT throw)
- Every request uses `Options(validateStatus: (_) => true)` and you branch on `resp.statusCode == 200`.
- `resp.data` for JSON endpoints is already-decoded (`List`/`Map`). Do not re-parse with `jsonDecode`.
- Prices/changes arrive as strings — parse with `double.parse(item['lastPrice'] as String)`.

## Algorithm (port EXACTLY — from getCryptoPrices)
```
getPrices(ids):
  fx = await getFx()
  result = <String,CryptoPrice>{}
  failedIds = <String>[]
  binanceIds = ids where id != 'USDT'
  for each id in binanceIds: validate against SYMBOL_REGEX; if invalid throw ApiError('invalid symbol: $id')

  if binanceIds not empty:
    querySymbols = '[' + binanceIds.map(id => '"${id}USDT"').join(',') + ']'
    url = 'https://api.binance.com/api/v3/ticker/24hr?symbols=' + Uri.encodeComponent(querySymbols)
    resp = dio.get(url, validateStatus all)
    if resp.statusCode == 200 and resp.data is List:
      for item in resp.data:
        id = (item['symbol'] as String) with trailing 'USDT' removed
        usd = double.parse(item['lastPrice']); chg = double.parse(item['priceChangePercent'])
        result[id] = CryptoPrice(usd, chg, usd*fx, chg)
      for id in binanceIds: if result[id]==null: failedIds.add(id)
    else:                      # batch failed → per-symbol
      for id in binanceIds:
        try:
          single = dio.get('https://api.binance.com/api/v3/ticker/24hr?symbol=' + Uri.encodeComponent(id+'USDT'), validateStatus all)
          if single.statusCode == 200:
            usd = double.parse(single.data['lastPrice']); chg = double.parse(single.data['priceChangePercent'])
            result[id] = CryptoPrice(usd, chg, usd*fx, chg)
          else: failedIds.add(id)
        catch: failedIds.add(id)

  for id in failedIds:               # Yahoo fallback (only if callback provided)
    if yahooFallback != null:
      try:
        y = await yahooFallback('${id}-USD')
        if y != null: result[id] = CryptoPrice(y.price, y.chg, y.price*fx, y.chg)
      catch: (ignore)

  for id in ids:                     # USDT manual
    if result[id]==null and id=='USDT':
      result['USDT'] = CryptoPrice(1, 0, fx, 0)

  return result
```

## Constants
- `SYMBOL_REGEX = RegExp(r'^[A-Za-z0-9.\-^=]{1,15}$')`

## Test vectors (`test/binance_client_test.dart`) — mocktail `class MockDio extends Mock implements Dio {}`
Register a fallback for `Options` in `setUpAll`: `registerFallbackValue(Options());`
Stub form: `when(() => dio.get(any(), options: any(named: 'options'))).thenAnswer((inv) async => Response(requestOptions: RequestOptions(path: ''), statusCode: 200, data: <the json>));`
Match batch-vs-single by URL substring with `any(that: contains('symbols='))` / `contains('symbol=')`.

1. **Batch success**: `getFx` returns 30. Batch call → 200, data =
   `[{'symbol':'BTCUSDT','lastPrice':'2350000','priceChangePercent':'1.5'}, {'symbol':'ETHUSDT','lastPrice':'120000','priceChangePercent':'-2.0'}]`.
   `getPrices(['BTC','ETH'])` → `result['BTC'].usd==2350000`, `usdChg==1.5`, `thb` closeTo(2350000*30, 0.01), `thbChg==1.5`; `result['ETH'].usd==120000`, `usdChg==-2.0`.
2. **Batch fails → per-symbol**: batch (`contains('symbols=')`) → statusCode 400; single (`contains('symbol=')` but not `symbols=`) for BTC → 200 `{'symbol':'BTCUSDT','lastPrice':'100','priceChangePercent':'0'}`.
   `getPrices(['BTC'])` → `result['BTC'].usd==100`.
3. **Yahoo fallback**: inject `yahooFallback: (s) async => (price: 5.0, chg: 1.0)`. Batch → 200 with empty list `[]` (so BTC becomes a failedId). `getPrices(['BTC'])` → `result['BTC'].usd==5.0`, `thb` closeTo(5*fx).
4. **USDT manual**: `getFx` returns 30. `getPrices(['USDT'])` (no Binance call needed). → `result['USDT'].usd==1`, `usdChg==0`, `thb==30`.
5. **Invalid symbol**: `getPrices(['BTC!!'])` throws `ApiError`.
