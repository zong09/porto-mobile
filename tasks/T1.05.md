## Spec — plain wrapper class
```dart
class TransactionDao {
  final AppDatabase db;
  TransactionDao(this.db);
  Future<void> upsert(TransactionsCompanion c) => db.into(db.transactions).insertOnConflictUpdate(c);
  Future<void> delete(String id) => (db.delete(db.transactions)..where((t)=>t.id.equals(id))).go();
  Future<List<Transaction>> byAsset(String assetId) =>            // oldest→newest by date
      (db.select(db.transactions)..where((t)=>t.assetId.equals(assetId))
        ..orderBy([(t)=>OrderingTerm(expression: t.date)])).get();
  Stream<List<Transaction>> watchByAsset(String assetId);         // same, .watch()
  Future<List<Transaction>> all();
}
```
`Transaction` = Drift generated row class.

## Tests
1. Insert 2 txs for asset a1 (dates 2026-01-02, 2026-01-01); `byAsset('a1')` returns them oldest-first (01 then 02).
