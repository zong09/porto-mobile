## LiabilityRepo (`repos/liability_repo.dart`)
```
const _ccy={'THB','USD'};
class LiabilityRepo { final LiabilityDao dao; LiabilityRepo(this.dao);
  Future<Liability> create({name, amount, currency}):
    if name.trim().isEmpty -> ArgumentError('name');
    if amount <= 0 -> ArgumentError('amount');
    if !_ccy.contains(currency) -> ArgumentError('currency');
    id=uuid; await dao.upsert(LiabilitiesCompanion.insert(id:id, name:name.trim(), amount:amount,
      currency: Value(currency)));
    return Liability(id:id, name:name.trim(), amount:amount, currency:currency);
  Future<void> save(Liability l):
    if l.name.trim().isEmpty -> ArgumentError('name'); if l.amount<0 -> ArgumentError('amount');
    if !_ccy.contains(l.currency) -> ArgumentError('currency'); await dao.upsert(l.toCompanion(true));
  Future<void> remove(String id) => dao.deleteLiability(id);

  // PORT of liabilities.service.ts#adjust:
  Future<void> adjust({liabilityId, type, amount, date}):
    if amount <= 0 -> ArgumentError('amount');
    if type != 'pay' && type != 'add' -> ArgumentError('type');
    await dao.db.transaction(() async {
      current = (await dao.all()).where((x)=>x.id==liabilityId).firstOrNull;
      if current == null -> throw ArgumentError('liability not found');
      next = (type=='pay') ? current.amount - amount : current.amount + amount;
      clamped = next < 0 ? 0.0 : next;                       // Math.max(0, next)
      await dao.addTx(LiabilityTransactionsCompanion.insert(id: const Uuid().v4(),
        liabilityId: liabilityId, type: type, amount: amount, date: date,
        createdAt: DateTime.now().toUtc().toIso8601String()));
      await dao.upsert(current.copyWith(amount: clamped).toCompanion(true));
    });
  Future<List<Liability>> all() => dao.all();
  Future<List<LiabilityTransaction>> txsFor(String id) => dao.txsFor(id);
}
final liabilityRepoProvider = Provider((ref) => LiabilityRepo(ref.watch(liabilityDaoProvider)));
```

## Test (`test/liability_repo_test.dart`)
`final db = AppDatabase(NativeDatabase.memory()); final repo = LiabilityRepo(LiabilityDao(db));`
`tearDown(() => db.close());`
1. `create(name:'Card', amount:1000, currency:'THB')` → `all()` has 1, amount==1000, currency=='THB'.
2. `create(name:'', amount:1, currency:'THB')` throws; `amount:0` throws; `currency:'EUR'` throws.
3. adjust pay: create amount 1000; `adjust(liabilityId:id, type:'pay', amount:300, date:'2026-01-05')`
   → `all()` amount==700; `txsFor(id)` has 1 tx (type 'pay', amount 300).
4. adjust add: from 700, `adjust(type:'add', amount:200, date:'2026-01-06')` → amount==900;
   `txsFor(id)` has 2 rows.
5. adjust clamp: amount 100; `adjust(type:'pay', amount:500, ...)` → amount==0 (not negative).
6. `adjust(type:'bogus',...)` throws; `adjust(amount:0,...)` throws.
7. `remove(id)` → `all()` empty AND `txsFor(id)` empty (FK cascade).
