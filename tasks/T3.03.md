## Goal

`<rect>` bars. Used for monthly buy/sell summaries and history bars.

## API to build
```dart
/// Vertical bars, evenly spaced. Positive bars grow up from the baseline;
/// if any value is negative, the baseline is mid-height and negatives grow down.
class BarChart extends StatelessWidget {
  final List<double> values;
  final Color barColor;
  final List<Color>? perBar;  // optional per-bar colors (length must match values)
  final double gap;           // px between bars, default 6
  final double radius;        // top-corner radius, default 4
  const BarChart(this.values, {super.key, required this.barColor, this.perBar,
      this.gap = 6, this.radius = 4});
  @override
  Widget build(BuildContext context) => CustomPaint(
      painter: BarChartPainter(values, barColor: barColor, perBar: perBar, gap: gap, radius: radius),
      child: const SizedBox.expand());
}

class BarChartPainter extends CustomPainter { ... }
```

## Algorithm
Let `n = values.length`, `w = size.width`, `h = size.height`.
1. If `n == 0` → paint nothing, return.
2. `maxAbs = max(values.map((v) => v.abs()))`; if `maxAbs == 0` → paint nothing (or flat), return.
3. `hasNeg = values.any((v) => v < 0)`; `baseline = hasNeg ? h/2 : h`.
   `avail = hasNeg ? h/2 : h` (max bar length from baseline).
4. `barW = (w - gap * (n - 1)) / n`.
5. For each i: `x = i * (barW + gap)`; `len = (values[i].abs() / maxAbs) * avail`;
   `top = values[i] >= 0 ? baseline - len : baseline`;
   rect = `Rect.fromLTWH(x, top, barW, len)`; draw as
   `RRect` with only the **outer** corners rounded by `radius`
   (top corners for positive bars, bottom corners for negative), fill with
   `perBar?[i] ?? barColor`.
6. `shouldRepaint`: true if `values`, `barColor`, `perBar`, `gap`, or `radius` differ.

## Test vectors (`test/bar_chart_test.dart`)
1. `testWidgets('five positive bars paint')`: pump
   `SizedBox(width: 200, height: 80, child: BarChart(const [3,7,5,9,4], barColor: Color(0xFFEC6530)))`
   in a `MaterialApp`; `expect(tester.takeException(), isNull)`.
2. `testWidgets('mixed sign bars paint')`: pump `BarChart(const [3,-2,5,-4], barColor: Color(0xFF1E9396))`; no exception.
3. `testWidgets('empty / all-zero paints nothing')`: pump `BarChart(const [], barColor: Color(0xFFEC6530))`
   and `BarChart(const [0,0], barColor: Color(0xFFEC6530))`; no exception.
4. `test('shouldRepaint true on values change')`:
   `expect(BarChartPainter(const [1,2], barColor: const Color(0xFFEC6530)).shouldRepaint(BarChartPainter(const [1,3], barColor: const Color(0xFFEC6530))), isTrue);`
