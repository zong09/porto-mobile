## CurrencyConverter (currency_converter.dart)
```
toThb(amount, currency, fx):   return amount * (currency=='USD' ? fx : 1)
convert(amount, from, to, fx): thb = toThb(amount, from, fx);
                               return to=='USD' ? thb / fx : thb
```
(currency strings are 'THB'|'USD'.)

## Formatters (formatters.dart) — use intl NumberFormat
```
money(v, {currency='USD'}):  2 decimals, thousands grouped, no currency symbol.
                             e.g. money(1234.5)=='1,234.50', money(0)=='0.00', money(-12.3)=='-12.30'
compact(v):                  1.2K / 3.4M / 1.0B style, 1 decimal; abs<1000 → money(v) 2dp.
                             e.g. compact(1200)=='1.2K', compact(3_400_000)=='3.4M', compact(950)=='950.00'
pct(v):                      signed, 2 decimals, % suffix. pct(1.234)=='+1.23%', pct(-0.5)=='-0.50%', pct(0)=='+0.00%'
```
