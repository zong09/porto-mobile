## Facts you need (do not go looking for them)
- `PortfolioAssetDao(AppDatabase db)` exists in `lib/src/db/portfolio_asset_dao.dart` with:
  `upsertPortfolio(PortfoliosCompanion)`, `allPortfolios()→Future<List<Portfolio>>`,
  `deletePortfolio(String id)`, `reorderPortfolios(List<String>)`,
  `upsertAsset(AssetsCompanion)`, `assetsFor(String portfolioId)→Future<List<Asset>>`,
  `deleteAsset(String id)`.
- Drift row **`Portfolio`**: fields `id`(String), `name`(String), `color`(int), `sortOrder`(int).
- Drift row **`Asset`**: `id`, `portfolioId`, `type`, `symbol`, `name`, `currency`(String),
  `cgId`(String?), `yahooSymbol`(String?), `manualPrice`(double?), `direction`(String),
  `sortOrder`(int). All are on the row as shown (enum columns are plain `String`).
- Companions: `PortfoliosCompanion.insert(id:, name:, color:, sortOrder: Value(int))` — `sortOrder`
  is optional (`Value<int>`). `AssetsCompanion.insert(id:, portfolioId:, type:, symbol:, name:,
  currency: Value('USD'), cgId: Value(x), yahooSymbol: Value(x), manualPrice: Value(x),
  direction: Value('long'), sortOrder: Value(n))` — everything except id/portfolioId/type/symbol/name
  is an optional `Value<>`. Drift rows have `.toCompanion(true)`.
- All classes above come from `import '../db/database.dart';`. Import `package:drift/drift.dart`
  (for `Value`), `package:uuid/uuid.dart`, `package:flutter_riverpod/flutter_riverpod.dart`,
  `package:drift/native.dart` (test only, `NativeDatabase.memory()`).
- Do **not** import `models.dart` (its freezed `Portfolio` would collide with the Drift row).

## PortfolioRepo (`repos/portfolio_repo.dart`)
```
class PortfolioRepo { final PortfolioAssetDao dao; PortfolioRepo(this.dao);
  Future<Portfolio> create({required String name, required int color}):
    n = name.trim(); if n.isEmpty -> throw ArgumentError('name');
    if color<0 || color>5 -> throw ArgumentError('color');
    order = (await dao.allPortfolios()).length; id = const Uuid().v4();
    await dao.upsertPortfolio(PortfoliosCompanion.insert(id:id, name:n, color:color, sortOrder: Value(order)));
    return Portfolio(id:id, name:n, color:color, sortOrder:order);
  Future<void> save(Portfolio p):
    if p.name.trim().isEmpty -> throw ArgumentError('name');
    if p.color<0||p.color>5 -> throw ArgumentError('color');
    await dao.upsertPortfolio(p.toCompanion(true));
  Future<void> remove(String id) => dao.deletePortfolio(id);
  Future<void> reorder(List<String> ids) => dao.reorderPortfolios(ids);
  Future<List<Portfolio>> all() => dao.allPortfolios();
}
final portfolioRepoProvider = Provider((ref) => PortfolioRepo(ref.watch(portfolioAssetDaoProvider)));
```

## AssetRepo (`repos/asset_repo.dart`)
```
const _types = {'crypto','th','us','fund','deposit'}; const _ccy={'THB','USD'}; const _dir={'long','short'};
class AssetRepo { final PortfolioAssetDao dao; AssetRepo(this.dao);
  void _validate(type, symbol, name, currency, direction, manualPrice):
    if name.trim().isEmpty -> ArgumentError('name'); if symbol.trim().isEmpty -> ArgumentError('symbol');
    if !_types.contains(type) -> ArgumentError('type'); if !_ccy.contains(currency) -> ArgumentError('currency');
    if !_dir.contains(direction) -> ArgumentError('direction');
    if manualPrice!=null && manualPrice<0 -> ArgumentError('manualPrice');
  Future<Asset> create({portfolioId, type, symbol, name, currency, cgId, yahooSymbol, manualPrice, direction='long'}):
    _validate(...); order=(await dao.assetsFor(portfolioId)).length; id=uuid;
    await dao.upsertAsset(AssetsCompanion.insert(id:id, portfolioId:portfolioId, type:type,
       symbol:symbol.trim(), name:name.trim(), currency:Value(currency), cgId:Value(cgId),
       yahooSymbol:Value(yahooSymbol), manualPrice:Value(manualPrice), direction:Value(direction),
       sortOrder:Value(order)));
    return Asset(id:id, portfolioId:portfolioId, type:type, symbol:symbol.trim(), name:name.trim(),
       currency:currency, cgId:cgId, yahooSymbol:yahooSymbol, manualPrice:manualPrice,
       direction:direction, sortOrder:order);
  Future<void> save(Asset a):
    _validate(a.type,a.symbol,a.name,a.currency,a.direction,a.manualPrice);
    existing = (await dao.assetsFor(a.portfolioId)).where((x)=>x.id==a.id).firstOrNull;
    if existing!=null && existing.currency != a.currency -> throw ArgumentError('currency is locked');
    await dao.upsertAsset(a.toCompanion(true));
  Future<void> remove(String id) => dao.deleteAsset(id);
  Future<List<Asset>> allFor(String pid) => dao.assetsFor(pid);
  Future<List<Asset>> all() => dao.allAssets();
}
final assetRepoProvider = Provider((ref) => AssetRepo(ref.watch(portfolioAssetDaoProvider)));
```

## Test (`test/portfolio_asset_repo_test.dart`)
Setup per test: `final db = AppDatabase(NativeDatabase.memory());` then
`final pRepo = PortfolioRepo(PortfolioAssetDao(db));` etc. `tearDown(() => db.close());`.
Cover:
1. `create(name:'Main', color:2)` → `all()` has 1, its name=='Main', color==2, sortOrder==0.
   Second create → sortOrder==1.
2. create with `name:'  '` throws `ArgumentError`; `color:6` throws `ArgumentError`.
3. `save(p.copyWith(name:'Renamed'))` → `all()` first name=='Renamed' (one row, not two).
4. `remove(id)` → `all()` empty.
5. `reorder([id2, id1])` → `all()` order is [id2, id1] (sortOrder 0,1).
6. AssetRepo `create(portfolioId:p.id, type:'crypto', symbol:'BTC', name:'Bitcoin', currency:'USD')`
   → `allFor(p.id)` has 1, currency=='USD', direction=='long', sortOrder==0.
7. Asset create with `type:'bogus'` throws; `symbol:''` throws.
8. currency lock: create asset currency 'USD'; `save(asset.copyWith(currency:'THB'))` throws
   `ArgumentError`; `save(asset.copyWith(name:'BTC2'))` succeeds (name changes, currency same).
9. FK cascade: create portfolio + asset under it; `pRepo.remove(portfolioId)` → `aRepo.all()` empty
   (foreign_keys pragma is ON in AppDatabase).
